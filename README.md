# 🚗 AI 차량 외관 손상 자가 진단 서비스 (MVP)

> **미래자동차공학과 4학년 캡스톤 디자인 프로젝트**
> 본 프로젝트는 자율주행 및 카셰어링 시대의 효율적인 차량 관리를 위한 AI 이미지 인식 기반 외관 진단 프로토타입입니다.

---

## 📝 프로젝트 설명
이 서비스는 사용자가 차량의 외관 사진을 업로드하면, AI 모델이 차량의 손상 여부(Dent, Scratch 등)를 스스로 진단합니다. 카셰어링 서비스의 반납 프로세스 자동화나 자율주행 차량의 유지보수 관리를 목적으로 합니다.

---

## ✨ 현재 구현된 핵심 기능 (v1.0.0)
- **차량 사진 업로드**: 사용자가 촬영한 차량 이미지를 웹 서비스에 업로드
- **AI 손상 분석 시뮬레이션**: 업로드된 이미지를 바탕으로 AI 모델이 손상 여부 분석
- **진단 결과 리포트**: 손상 감지 유무 판정 및 신뢰도 출력

---

## 🛠 실행 방법 (How to Run)
1. 필수 라이브러리 설치: `pip install streamlit pillow`
2. 서비스 실행: `streamlit run app.py`
